**Инструмент командной строки преобразует конфигурационные файлы, написанные на учебном конфигурационном языке, в стандартный формат YAML**


Поддерживает:     
-*обработку многострочных комментариев*   
-*обработку чисел*   
-*обработку массивов*  
-*обработку словарей*  
-*обработку имён*  
-*обработку констант с вычислением константного выражения*   
-*вложенные конструкции*   


Класс ConfigParser осуществляет парсинг конфигурационного файла  
**Метод** __init__(self) инициализирует парсер  
**Метод** parse(self, text) основной метод парсера  
**Метод** parse_file(sel, filename) читает и парсит файл  
**Метод** parse_single_value(self, value_str, parse_constants) основной парсер одного значения  
Остальные методы нужны для стандартизации синтаксиса, работы с константами, массивами, словарями   
**Функция main** запускает преобразование  


*Команда для сборки и запуска тестов:* py config.py -i входной_файл.conf -o выходной_файл.yaml  


*Примеры использования:* настройка ci-cd пайплана, конфигурация веб-сервера
